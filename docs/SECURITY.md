# Politica de seguridad

## Reportar vulnerabilidades

Por favor, no reportes vulnerabilidades en issues publicos.

Usa GitHub Security Advisories para reportes privados:
- Ve a la pestaña "Security" del repo
- Crea un "Security advisory"
